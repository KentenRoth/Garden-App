What's Planted?



Goals/ideas for this app (in no specific order)

  1. Login feature to allow user to save outside of local storage
  2. Display days until harvest
  3. Add how many plants are planted
  4. Display possible pests associated with attacking plant
  5. Display planting distance and depth
  6. If plant needs full sun or should be planted in shade of other plants
  7. Watering schedule
  8. Search for API for info on numbers 4-7
